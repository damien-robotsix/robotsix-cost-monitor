# Security Policy

## Supported Versions

Version 0.1.x is currently being supported with security updates.

## Reporting a Vulnerability

Please do not open public issues for security vulnerabilities. Instead, report them privately via one of the following methods:

1. GitHub's private vulnerability reporting tool at https://github.com/damien-robotsix/robotsix-cost-monitor/security/advisories/new
2. Email the maintainer at **damien.robotsix@gmail.com**

You will receive an acknowledgement within 72 hours. After that, the maintainer will work with you to understand the issue, develop a fix, and coordinate a disclosure timeline.
